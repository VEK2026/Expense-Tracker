# Название проекта "Currency Converter" (Конвертер валют)
## Автор: Гоголева Полина
### Описание программы
Кратко: Графическое приложение для отслеживания личных расходов с фильтрацией, подсчётом суммы за период, JSON и Git 

## Функционал
### При открытии программы заполните форму с полями:
○ Сумма
○ Категория (например, еда, транспорт, развлечения)
○ Дата

### Кнопка «Добавить расход» добавляет запись в таблицу.
Подсчёт суммы расходов происходит автоматически за выбранный период.
## Возможна фильтрация:
○ По категории
○ По дате

Требования для запуска
Что нужно установить на компьютер: - Python 3.10 или выше - Библиотеки: pip install matplotlib requests ---
